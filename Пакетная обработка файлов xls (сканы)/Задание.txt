Скрипт 1.
Проверка корректности данных
Исходные данные представляют собой папку с экселевскеми документами содержащими распознанные остканированные таблицы (см 100 примеров)
Для меня интерес представляются 5 столбцов таблицы (Номер	Методика ОИ Показатель Диапазон)
Столбцы в разных файлах могут быть смещены, определяются по наличию ключевых слов (см файл Настройки  
 лист Опознавание столбцов)
Программа проверяет корректность данных и ячейки с некорректными данными выделяет цветовой заливкой.

Проверяет 
0. Перед работой сбрасывает цветовое выделение открытого файла если оно есть.
1. Наличие пустых ячеек в номерах, методиках, ОИ, показателях, диапазоне - если находит выделяет желтой заливкой.
2. Наличие номеров и их корректность (там числа и числа по-порядку), если не в порядке выделяет красной заливкой неверные номера. Нюанс в ряде таблиц есть шапки с номерами столбцов (1,2,3,4,5) их нужно игнорировать.
3. Наличие "Красные символы" в определенных столбцах и если находит какие-то из них выделяет ячейку красной заливкой. Для каждого столбца свой набор Красных симоволов (см лист Красные символы)
4. Для каждого из столбов (методики, ОИ, показатели, диапазоны)  определен  словарь слов (см лист Словари)
 если в данных встречает слова которых нет в словаре, то 1)выделяет ячейки с новыми словами в исходных файлах желтой завливкой 2) добавляет новые уникальные слова в нужный столбец на листе Найденные новые слова. Внимание! Только уникальные слова, если слово одно слово уже добавлено один раз в такой то столбце на листе Найденные новые слова второй раз добавлять туда это слово не нужно, но ячейка в исходном файле все равно выделяется желтой заливкой.
4.1. Для столбца диапазоны свой нюанс: числа игнорируются, словарь составляется только для единиц измерения
4.2. Для столбца методики нюанс: номера методик игнорируются, словарь составляется только для частей их буквенных обозначений

Для более корректного разделения слов в Настойках есть лист Символы разделители слов который определяет символы которые являются разделителями слов для каждого столбца. (Это сделано потому что например в показателях нормальным может быть название показателя "3.4-Бензпирен" или "бенз(а)пирен", в то время как в других столбцах скобки и тире могут быть разделителями слов)

Ведется лог (лист Лог)
В логе отмечается открытие и закрытия файла
Количество найденных новых добавленных слов в каждую категорию по каждому файлу
Сообщения об ошибках

Работает с пакетом экселевских файлов (папкой)

В прикрепе набор примеров.
И образец листа настроек

Скрипт 2
Пакетная замена.
Запускается отдельной кнопкой, работает с тем же пакетом файлов (допустим отдельный выбор папки с которой будет работать)
Используя лист замен (см лист Замены)
осуществляет поиск нужных слов и словосочетаний в нужном столбце и заменяет найденное слово или словосочетание на нужное.
